## Getting Started

### Prerequisites
* npm
    ```sh
    npm install npm@latest
    ```

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/jduriu/clubsport-calendar
    ```
2. Navigate to the root folder of the cloned application
3. Install NPM packages
    ```sh
    npm install
    ```
4. Run the project in development mode
    ```
    npm run dev
    ```
